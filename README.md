# InsightStay

AI-powered guest feedback intelligence platform.

## Overview

InsightStay is a full-stack web application that helps hospitality businesses analyze guest reviews, identify customer sentiment, detect recurring experience themes, and gain actionable insights from feedback. The platform provides an intuitive dashboard for managing guest reviews while leveraging Artificial Intelligence to improve customer engagement and service quality.

---

## Features

- AI-powered guest review analysis
- Sentiment classification (Positive, Neutral, Negative)
- Guest review management (Create, Read, Update, Delete)
- Search reviews by guest, hotel, review text, or sentiment
- Interactive analytics dashboard
- Responsive user interface with Dark Mode
- RESTful API integration
- Persistent cloud database using MongoDB Atlas
